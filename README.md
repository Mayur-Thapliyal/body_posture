# body_posture
Project to correct your yoga posture



#### Setup:
- Clone the repo and create a virtual python env  (recommended) by using following command:-

``` bash
python3 -m venv venv
source venv/bin/activate
```

- After creating and activating your env run:-
```bash
pip install -r requirements.txt 
```

#### Running Project
- Run
```python
python3 main.py
```
This will run your code in your local camera by default you can change the port by passing your camera IP in code.
